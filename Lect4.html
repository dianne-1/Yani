1. Hyperlinks (<a> Tag)

Links allow users to jump between documents. The href (Hypertext Reference) attribute specifies the destination.

Example Link Types :

  • Absolute: <a href="http://google.com">Google</a>
  • Relative: <a href="template.htm">template</a>
  • Parent Directory: <a href="../index.html">Home Page</a>
  • Subdirectory: <a href="hw1/business.html">Business Page</a>

Target Attribute:

Defines where the linked document opens:

<a href="https://www.w3schools.com/" target="_blank">Opens in New Tab</a>
<a href="template.htm" target="_top">Opens in Full Window</a>

2. Understanding Image Formats

Choosing the right format depends on the content and performance needs .

AVIF: Next-gen format; 50% smaller than JPEG with superior compression .
WebP: Created by Google; supports lossy/lossless compression and transparency.
GIF: Best for navigation buttons and animations; limited to 256 colors.
JPEG: Excellent for photorealistic digital images .
PNG: Supports alpha channels (65,000+ layers of transparency).

3. Image Syntax & Examples

Images are "empty" tags (no closing tag) using the src and alt attributes .

Basic Implementation:

<img src="panda.png" alt="Panda Bear" style="width:500px;height:600px;">

Images from Different Sources :

  • Subfolder: <img src="images/circle.jpg" alt="Circle">
  • External Server: <img src="https://www.ue.edu.ph/.../ccss.png" alt="UE-CCSS">

Image as a Link :

<a href="template.html">
  <img src="Rudolph.jpg" alt="rudolph" style="width:42px;height:42px;border:0;">
</a>

Floating Images:

Use style="float:right;" or style="float:left;" to wrap text around an image.

4. Image Maps ("Hot Spots")

An image map allows multiple clickable areas on a single image.

Coordinate Syntax:

Shape Coordinates (coords)
rect "x1, y1, x2, y2" (Top-left & Bottom-right corners)
circle "x, y, r" (Center point and radius)
poly "x1, y1, x2, y2, x3, y3..." (Each corner of the polygon)
<map name="example-map">
  <area shape="rect" coords="0,0,50,50" href="link1.html" alt="Top Left">
</map>
