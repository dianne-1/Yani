1. What is Bootstrap?

Bootstrap is the most popular HTML, CSS, and JS framework for developing responsive, mobile-first websites.

  • Origin: Created by Mark Otto and Jacob Thornton at Twitter (2011).
  • Mobile-First: Designed so styles are optimized for mobile devices by default.
  • Uniformity: Provides a clean and consistent interface for developers.

2. Containers

Containers are the most basic layout element in Bootstrap. They are required when using the grid system.

.container

Provides a responsive fixed width container.

.container-fluid

Provides a full width container, spanning the entire width of the viewport.

3. Contextual Classes (Colors)

Bootstrap provides classes to convey meaning through color.

Meaning Text Class Background Class
Important / Blue .text-primary .bg-primary
Success / Green .text-success .bg-success
Danger / Red .text-danger .bg-danger
Warning / Yellow .text-warning .bg-warning
Information / Teal .text-info .bg-info

4. Typography & Tables

Bootstrap styles HTML headings and tables to look better automatically.

  • Display Headings: Use .display-1 through .display-4 for larger, stand-out titles.
  • Small Text: Use the <small> element to create lighter, secondary text.
  • Striped Tables: Add .table-striped to the <table> tag for zebra-striping.
